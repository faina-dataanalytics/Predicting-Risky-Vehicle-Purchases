# Predicting-Risky-Vehicle-Purchases
Dieses Projekt entwickelt ein ML‑Modell zur Vorhersage von „Bad Buys“ bei Autoauktionen. Es umfasst Datenaufbereitung, Feature Engineering, Modelltraining und Evaluierung mit Fokus auf hohem Recall. Die Pipeline ist vollständig reproduzierbar und enthält einen geschäftsorientierten Abschlussbericht.
